Principais motivos para eu gostar deles

  • Adoro a steam, pois junta uma gama enorme de games para o computador, com um preço menor do que o mercado ou a mídia física. Além do que, sendo mídio online, tenho garantia de que os CDs dos games não vão estragar com o tempo, ou arranhar, etc...
  • O W3SCHOOLS é ótimo, está me ajudando bastante para aprender alguns detalhes bônus sobre HTML e CSS, coisas que não temos tempo de aprender em aula.
  • A Wiki do Final Fantasy é um repositório enorme de informação sobre todos os games da série, como eu amo Final Fantasy, não teria como não adorar este site!
